## Simple task list

### Requirements

  * PHP ^7.2.5
  * Composer
  * Node.js & npm
  * PostgreSQL 
  * SQLite for test

### Setup

```sh
$ make setup
```

### Run

```sh
$ make start
```
### Tests

```sh
$ make test
```
